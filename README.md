# Port Congestion Prediction

Distributed Deep Learning based Port Congestion Prediction using YOLOv8 and Multi-Task CNN.

## Features

- Ship Detection using YOLOv8
- K-Means based Congestion Labeling
- Port Capacity Estimation
- Weather Aware Preprocessing
- Multi-Task CNN
- Distributed Deep Learning

## Dataset

- Airbus Ship Detection Dataset

## Results

- Precision: 0.77
- Recall: 0.64
- mAP50: 0.72
- mAP50-95: 0.45

## Technologies

- Python
- PyTorch
- YOLOv8
- Scikit-Learn
- OpenCV
